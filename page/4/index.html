Mac OS 软件推荐列表

Published on: Nov 18, 2014
Tags:

我没有离开，我还在，还好我还在。

用 Mac OS X 已经半年多了，总想总结总结，就把我的纠结与不纠结都纠结的写在这里吧，给自己的一个理由，已经折腾过的理由。

Macbook 真的很适合科研和学习使用，至少对于我这样的理工科的很适合。以下的推荐不仅仅是我自己的摸索也参考了诸多网站和大神的列表。国内用的比较多的网站就是 MacGG 了，不要和我说版权问题，相信很多人和我一样，买软件之前先用用 x 版的试用下。

学术类

  • Office 2011:可能出现不兼容其他版本的情况，但是，不管再怎么不兼容还是要有的软件。（而且，我感觉也不错的用着）

  • Papers:一个学术论文管理、发掘和论文参考文献辅助工具。有学生折扣。（当然，其他的类似软件也都可以，但是我认为无论是哪一个软件，一定要坚持使用上三个月之后，有一定的自己数据的积累，这样才会慢慢对你的需求明确起来）

  • OmniGraffle:一个画示意图、结构框图、原理图的矢量工具。价格有点贵，但是功能的确很优秀。

  • DataGraph:做出专业曲线图、分析结果图、统计分布图等的数据可视化工具。也有点贵，图的效果还是很专业的。另外，如果希望使用免费工具的话，可以选择Plot2。但是两个工具的学习曲线还是有一点区别的。

  • LaTeXiT 和 MathType:因为做论文可能会用到 Office 2011 和 LaTeX 两种平台，所以这两个公式编辑器还是都准备上。

  • Seashore:很喜欢的一个免费画图小工具，可以在实验结果图上做一些简单的标注，功能不多，但简单实用。

有的狗为什么把碗舔的那么干净？

Published on: Aug 17, 2014
Tags:

这是一个令人悲伤的故事，自从狗儿们被称为狗，它们就已经变成了人类的朋友。一同抵御外敌、相依为命。可是有一天人们不再需要他们的保护了，狗的就有了其他的身份：食、色。它们没有选择离开人类回归自然，不是因为它们回归自然后生存不下去（我见过的狗基本上都属于杂食，吃草估计也可以生存），也不是因为它们没有这个智商，而是它们选择了陪伴。这是一个客观的原因。狗的祖祖辈辈都是跟人类一起生活着，虽然有过流浪，有过被捕杀和驱逐，但是它们都是围绕在人的周围。如果真的像科幻电影里的那样有一天《人猿崛起》，我更加可能相信的是崛起的主角是狗们。因为狗与人类共同生活了这么久，相比猿类狗比他们更加了解人类。还好，直到今天狗还是人类的好朋友，可能关系更加好了。

那么，为什么有的狗把它吃饭的碗舔的那么干净呢？在我看来它是为了表示自己没吃饱。当喂它的人看到被舔干净的碗的时候就会给它更多一点食物。原因很简单。但是说好的悲伤呢？说好的故事呢？

悲伤的是，我们很多人不也是一只愿意把碗舔干净的狗么？我们时刻生存在一个人与狗的世界里。好，还是不好？

Install NCL/NCAR on Mac OS X

Published on: Jul 23, 2014
Tags:

The NCAR Command Language (NCL) is the coolest graphics tool I’ve ever seen. NCL has robust file input and output. It can read and write netCDF-3, netCDF-4 classic, netCDF-4, HDF4, binary, and ASCII data. It can read HDF-EOS2, HDF-EOS5, GRIB1, GRIB2, and OGR files. Though I don’t know most of the format above, but I saw its perfect plot on other one’s paper, so I decide to install it on my Macbook with Mac OS X 10.9.4.

Please look into this site and finish reading it before starting to step into the door.

Official site

I preferred to run the pre-compiled version, so the only problem after I installed according to the official guideline is the version of my GCC. Then I need to update GCC on my Mac, since I already have homebrew installed, brew install gcc48 will install the new version by itself. It takes about 30 minutes to compile from the source. Be patient!

The most complex work is setting some environmental variables, nothing special, just type what are recommended by the guideline. You won’t miss it.

如何在iBook中复制较长内容

Published on: Jul 9, 2014
Tags:

最近在看「囚徒健身」，因为相摘抄一点笔记所以才发现原来iBook为了保护版权和避免抄袭，限制了一次能复制的文本内容（多少个字的限制不记不清了），这种机制是好的，但是并不是所有的情况都适合，经过搜索终于找到一个能够在iBook里复制较多文本的办法如下：

  • 选中要复制的段落
  • 在Context Menu里找到More下面的Search Wikipedia
  • 然后在Safari里或者你的浏览器里就可以看到一个搜索文本框里的文本了
  • 复制 or 剪切里面的文本到你想放的地方就ok了

畅享Macbook下阅读的快乐吧。
