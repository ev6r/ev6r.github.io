使用Visual Studio不可不知的快捷键（持续更新）

Aug 26, 2019

Visual Studio常用快捷键

  1. Ctrl + -：回到上一次的位置，向后导航。
  2. Alt + Enter：打开属性面板。
  3. Ctrl + Shift + F：在文件中查找。

Visual Assist X常用快捷键

现在看来C或C++编程已经是一门很古老的技术了，但是在程序设计的进展中，它们一直褒有勃勃的生命力。本文将收集Visual Assist X（VAX）的常用快捷键，让你在写代码时快的飞起，重要的是跟上你的思维，提高代码质量。

VAX菜单和工具栏中最常用的快捷键如下：

  1. Alt + G：Goto Implementation 转到实现的位置，通常会在查看一个函数的实现时使用，可以做到快速跳转。
  2. Alt + /：自动生成成员函数的Doxygen化注释，需要手动修改VAX的脚本与VS的全局快捷键。
  3. Alt + Shift + F：查找所有引用，会在下方显示当前函数被调用的所有地方。
  4. Alt + O：在头文件和源文件之间进行切换，非常常用。

VAX脚本的Doxygen化注释的快捷键可以customize定制，定制方法见《Doxygen注释模板之Visual Assist X脚本》一文，个人定制的快捷键如下：

  1. /f：文件头
  2. /c：类
  3. /s：结构体
  4. /e：枚举
  5. /m：函数
  6. /：普通note
  7. //：成员变量
  8. ///：分割线
  9. /*：注释块
